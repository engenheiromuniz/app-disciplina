📘 About the Project

This project was developed for study purposes, integrating a backend in Spring Boot with a frontend in Angular 19.

The goal is to practice CRUD concepts, code organization, and fullstack development best practices.

🛠️ Technologies Used

  • Spring Boot (Java)
  • JPA / Hibernate
  • Relational Database
  • Angular 19 (Standalone Components)
  • SCSS for styling

🚀 How to Use the System

  • You can register a new discipline ✏️
  • You can list all disciplines 📚
  • You can update existing disciplines 🔄
  • You can delete disciplines 🗑️

ℹ️ Note

This project is intended for learning and practice. It is not a commercial application, but rather an exercise in integrating frontend and backend technologies.
