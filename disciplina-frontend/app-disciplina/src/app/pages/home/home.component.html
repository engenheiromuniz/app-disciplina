📘 Projeto de Estudo - Spring + Angular

Sobre o Projeto

Este projeto foi desenvolvido para fins de estudo, integrando um backend em Spring Boot com um frontend em Angular 19.

O objetivo é praticar conceitos de CRUD, organização de código e boas práticas de desenvolvimento fullstack.

Tecnologias Utilizadas

  • Spring Boot (Java)
  • JPA / Hibernate
  • Banco de Dados Relacional
  • Angular 19 (Standalone Components)
  • SCSS para estilização

Nota

Este projeto é apenas para aprendizado e prática. Não se trata de uma aplicação comercial, mas sim de um exercício de integração entre frontend e backend.
