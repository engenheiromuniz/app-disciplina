# 📘 Full Stack Study Project - Spring Boot + Angular 19

## 🔎 About the Project
This project was developed for **study purposes**, integrating a backend in **Spring Boot** with a frontend in **Angular 19**.  
The main goal is to practice **CRUD operations**, code organization, and fullstack development best practices.

---

## 🛠️ Technologies Used
- ⚙️ **Spring Boot (Java)**
- 🗄️ **JPA / Hibernate**
- 🛢️ **Relational Database**
- 🌐 **Angular 19 (Standalone Components)**
- 🎨 **SCSS for styling**

---

## 🚀 Features
- ✏️ **Register** a new discipline  
- 📚 **List** all disciplines  
- 🔄 **Update** existing disciplines  
- 🗑️ **Delete** disciplines  

---

## 📂 Project Structure
- **Backend**: Spring Boot application with JPA/Hibernate for persistence.  
- **Frontend**: Angular 19 application using standalone components and SCSS for styling.  
- **Database**: Relational database integrated with JPA/Hibernate.  

---

## ▶️ How to Run
1. **Backend**  
   - Navigate to the Spring Boot project folder.  
   - Run:  
     ```bash
     ./mvnw spring-boot:run
     ```
   - The backend will start on `http://localhost:8080`.

2. **Frontend**  
   - Navigate to the Angular project folder.  
   - Run:  
     ```bash
     npm install
     ng serve
     ```
   - The frontend will start on `http://localhost:4200`.
